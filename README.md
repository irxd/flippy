## Getting Started

### Installation

* Install dependency
   ```sh
   npm install
   ```
   or
   ```sh
   yarn install
   ```
* Run app
   ```sh
   npm start
   npm run android
   ```
   or
   ```sh
   yarn start
   yarn android
   ```
